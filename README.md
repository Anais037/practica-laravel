# Pràctica amb Laravel

## Descripció
Projecte inicial amb Laravel on configuro rutes, controladors i vistes. Inclou instal·lació, configuració i proves bàsiques. Aquest projecte m’ha servit per entendre l’estructura del framework i com funciona el MVC.

## Tecnologies utilitzades
- Laravel
- PHP
- Composer

## Com executar
1. Clonar el repositori
2. Instal·lar dependències amb `composer install`
3. Crear el fitxer `.env` i configurar la base de dades
4. Executar `php artisan serve` per iniciar el servidor
